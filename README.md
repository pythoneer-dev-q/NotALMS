## Запуск (локально)

1. Клонируй репозиторий
2. Установи зависимости:
   ```bash
   pip install fastapi uvicorn motor python-dotenv passlib[bcrypt] python-jose[cryptography] slowapi
   ```
3. Создай файл `.env` в корне (пример ниже)
4. Запусти MongoDB локально или используй Atlas
5. Запусти сервер:
   ```bash
   uvicorn front.server.main:app --reload
   ```
6. Открой в браузере: http://127.0.0.1:8004/index.html или /

## API (на данный момент)

Все эндпоинты под префиксом `/v1`

### Аутентификация
- `POST /v1/register` — регистрация (возвращает JWT)
- `POST /v1/login` — вход (возвращает JWT)
- `GET /v1/me` — данные текущего пользователя (Bearer токен в Authorization)

### Задания по ДНК/РНК
- `POST /api/generate` — генерирует задание
  ```json
  {"mode": 1|2|3, "length": 15}  // length опционально
  ```
- `POST /api/check` — проверяет ответ пользователя
  ```json
  {
    "mode": 1|2|3,
    "original_data": {...},     // то, что пришло из /generate
    "user_answer": "5'-АУУУАЦГГУГГЦАААГГЦ-3'"
  }
  ```

### Режимы заданий
1. мРНК по нижней (матричной) цепи ДНК
2. мРНК по верхней (матричной) цепи ДНК
3. Построить комплементарную цепь ДНК (антипараллельно)

Проверка учитывает:
- концы 5'/3'
- комплементарность
- антипараллельность
- длину
- лишние символы
