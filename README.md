Описание проекта

Проект представляет собой API-сервис на FastAPI для генерации и проверки заданий по ДНК и транскрипции.
API предназначено для учебной платформы и поддерживает три различных режима работы, каждый из которых соответствует отдельному типу задания.

Сервис умеет:
	1.	Генерировать цепи ДНК или задания на транскрипцию.
	2.	Проверять ответы пользователя на корректность.
	3.	Возвращать подробные ошибки: неправильные нуклеотиды, неправильные концы цепи, ошибки направления и др.

Каждый режим имеет свои отдельные эндпоинты для генерации и проверки.

⸻

Основные режимы

Всего предусмотрено три режима:

Режим 1

Транскрибируемая цепь — нижняя.
Пользователю выдаются две цепи ДНК. Задача — по нижней цепи (матрице) построить мРНК.

Режим 2

Транскрибируемая цепь — верхняя.
Пользователь получает две цепи ДНК. Задача — по верхней цепи (матрице) построить мРНК.

Режим 3

Создание комплементарной ДНК-цепи.
Пользователь получает только одну цепь ДНК и должен построить вторую, комплементарную и антипараллельную.

⸻

Эндпоинты API

Генерация заданий

POST /generate/mode1
POST /generate/mode2
POST /generate/mode3

Каждый из этих эндпоинтов возвращает JSON с исходными данными для задания и текстовой инструкцией.

Проверка ответов

POST /check/mode1
POST /check/mode2
POST /check/mode3

Эндпоинты принимают:
	•	исходные данные задания (те же, что были возвращены при генерации),
	•	ответ пользователя.

Ответ API содержит:
	•	признак корректности,
	•	список ошибок (если есть),
	•	правильный ответ.

⸻

Формат ошибок

API поддерживает несколько типов ошибок:
	•	wrong_end — неверно указаны концы цепей (5’ и 3’).
	•	wrong_direction — ошибка направления записи.
	•	wrong_base — несовпадение нуклеотидов по комплементарности.
	•	wrong_length — неверная длина цепи.
	•	extra_chars — посторонние символы или форматирование.
	•	other — прочие ошибки.

Каждая ошибка описывается структурой:

{
  "type": "wrong_base",
  "index": 7,
  "expected": "Г",
  "got": "А"
}

Основные функции

Сервис использует следующие группы функций:

Генерация
	•	генерация случайной цепи ДНК;
	•	построение комплементарной ДНК-цепи;
	•	построение мРНК на основе матричной цепи ДНК;
	•	нормализация формата записи цепей.

Проверка
	•	проверка концов цепи (5’ и 3’);
	•	проверка направления записи;
	•	проверка соответствия нуклеотидов;
	•	поиск и подсветка ошибок.

⸻

Формат цепей

Для всех режимов используется стандартный формат записи:

5'-АТТГАЦ...-3'
3'-ТААЦТГ...-5'

Для мРНК используется урацил (У) вместо тимина (Т).

Установка и запуск

Примерный процесс установки (может изменяться в зависимости от структуры проекта):

pip install -r requirements.txt
uvicorn app.main:app --reload

После запуска приложение будет доступно по адресу:

http://localhost:8000

Назначение проекта

Проект используется как часть учебной платформы (LMS).
На его основе строятся интерактивные задания, тренажёры и теория по теме «ДНК и транскрипция».

Если требуется, можно расширить функциональность, добавить визуализацию ошибок или интеграцию с фронтендом.
